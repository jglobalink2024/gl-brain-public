## Tier 1 — Crown Jewel Claims (95%+ novelty)

**1. Metabolic Clock**
Cross-vendor agent fatigue detection. Monitors Claude + GPT + Perplexity simultaneously for performance degradation; triggers cooldown/routing shifts when vendor output quality decays. Phase 3A.
**NOTE:** Gemini Deep Research (260413) flagged prior art risk on current claim language. Must be narrowed to specific data structures and GUI interventions before filing.

**2. Learned Agent Cards**
Dynamic runtime-learned capability profiles. Learns which agent performs best at which task type from actual performance data — not static config. Cross-vendor by definition. Phase 3D.
**NOTE:** Same Gemini flag — claim language under revision.

**3. Contradiction Radar**
Semantic conflict detection across parallel LLM tasks. Detects contradictory outputs from different vendors in real time. Zero confirmed prior art as of 260413. Phase 3C.
**NOTE:** Same Gemini flag — claim language under revision.

**All three crown jewel claims hinge on the cross-vendor boundary. No single vendor can replicate them by definition.**

## Terminology replacements (hard rules in all docs)

| Killed term | Use instead |
|------------|-------------|
| Decision Provenance Graph | Workspace Provenance Ledger (WPL) |
| Cognitive Arbitrage | Cross-agent synthesis gap / coordination loss |
| Cognitive Arbitrage Detection | Cross-Agent Synthesis Engine (CASE) |
| Institutional Memory Crystallization | Workspace Dream Cycle (WDC) |
| Light Sleep / REM / Deep Sleep | Phase 1 (Ingestion) / Phase 2 (Synthesis) / Phase 3 (Promotion) — patent docs only |
| MACP (broad combination claim) | Reference specific surviving claims by name |

**Note:** Biological metaphors (light sleep, REM, etc.) are still OK in marketing. Remove from patent claims only.

**Why:** Broad or preempted claims waste money and create prosecution history that limits future narrower claims. The cross-vendor boundary is the defensible moat — all claims must hinge on it.
**How to apply:** Never write "MACP combination claim" in new docs. Never use killed terms in patent or IP docs. All three crown jewel claims need attorney review before any public disclosure.
