## Positioning statement (locked)

"The cockpit, not the engines." Vendor-neutral Agent Operations Center. Eliminates human copy-paste relay between AI tools from different vendors.
Framing: "Vercel above Next.js" — COMMAND sits above CrewAI/LangGraph, not replacing them.

## Structural moat — triple kill test

Every confirmed competitor operates within a single framework or vendor. COMMAND's cross-vendor boundary is structural, not feature-based. No single vendor (Anthropic, OpenAI, Google) can build a cross-vendor coordination layer — each only controls their own ecosystem.

**COMMAND sits alone in this 2x2:**
- X-axis: Claude-locked → Vendor-agnostic
- Y-axis: Developer tool → Operator tool
- CMA, OpenAI Agents = vendor-locked + developer tool
- CrewAI, LangGraph = framework-locked + developer tool
- AgentCenter = OpenClaw-locked + developer tool (terminal required)
- **COMMAND = vendor-agnostic + operator tool (alone in quadrant)**
